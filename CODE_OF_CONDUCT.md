# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be professional, be respectful, critique work rather than people,
and assume good faith. Actuarial correctness arguments are welcome — heated,
even — but they are about the numbers and the sources, never about the person.

Report unacceptable behavior privately to the maintainer via GitHub
(profile: [@yerromnitsuj](https://github.com/yerromnitsuj)). Reports are
handled confidentially. The maintainer may warn, moderate, or ban accounts
whose behavior violates this code, at their discretion.
