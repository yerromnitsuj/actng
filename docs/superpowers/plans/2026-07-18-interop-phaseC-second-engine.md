### Task C1: sidecar v1 (interop/sidecar/)
FastAPI app per spec 7 EXACTLY: POST /v1/run/{method} with { triangles: { primary, secondary? }, selection?, exposure?, parameters?, seed? } → MethodResultDoc | StochasticResultDoc; GET /v1/engine (name/version/profiles: deterministic-cl, mack1993-vw, odp-bootstrap-distribution); GET /v1/health. Methods: MackChainladder, Chainladder, BornhuetterFerguson, Benktander, CapeCod, ClarkLDF, BootstrapODPSample (seeded via numpy random_state), MunichAdjustment (requires secondary; refused otherwise with a schema'd error). Consumes the actuarial-interchange Python package for parsing/authoring (bridges + extract_result). SE-less rule enforced (Mack on value-only selection → warning'd SE-less result or 422 per strictness param). Bearer-token auth (SIDECAR_TOKEN env; health unauthenticated), request-size limit, stateless (no persistence; engagementRef opaque passthrough into result extensions). Containerfile (Dockerfile) pinning python:3.12 + chainladder==0.9.2 + pandas; but ALL tests run against the app in-process via fastapi TestClient (no Docker needed locally; the image is CI/deploy packaging). requirements.txt pinned. Tests: per-method golden runs on the committed conformance fixtures (results match the committed clpy-*.json where profiles align), auth 401, Munich slot refusal, seeded bootstrap determinism, engagementRef passthrough, no-tenant-fields schema enforcement.
